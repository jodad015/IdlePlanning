# Questions

Quick async questions between Kodad and Zach. Lighter weight than a proposal — use these when you want the other person's take on something before you invest in fleshing it out.

## File Format

`NNN-short-description.md` (e.g., `001-how-many-skills-at-launch.md`)

## Lifecycle

- **Open** — Waiting for a response.
- **Answered** — Got a response. If it leads to a bigger conversation, spin it into a proposal.
- **Resolved** — No further action needed.

## Template

```markdown
# [Number] - [Title]

**Asked by**: [Name]
**Date**: YYYY-MM-DD
**Status**: Open

## Question

What do you want to know?

## Context

Any background that helps the other person answer well.

## Responses

### [Name] - [Date]
Response here.
```
