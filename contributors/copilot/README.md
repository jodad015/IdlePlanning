# Copilot's Scratch Space

Durable brainstorms from Copilot. A player prompts Copilot elsewhere and pastes the output here with a brief note about the prompt that generated it.

Useful for:
- Capturing Copilot's take on a specific mechanic or system
- Outside-perspective counterpoints to ideas the team is rallying around
- Reference material Copilot pulled together on related games or design patterns

Not a synthesis space — that's `design/`. This is raw AI contribution, clearly attributed to Copilot.
