# Gemini's Scratch Space

Durable brainstorms from Gemini. A player prompts Gemini elsewhere (voice mode, chat, etc.) and pastes the output here with a brief note about the prompt that generated it.

Useful for:
- Capturing Gemini's take on a specific mechanic or system
- Voice-mode brainstorm sessions that produced ideas worth keeping
- Outside-perspective counterpoints or reference material

Not a synthesis space — that's `design/`. This is raw AI contribution, clearly attributed to Gemini.
